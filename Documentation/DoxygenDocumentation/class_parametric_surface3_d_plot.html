Descripción detallada

Clase que permite representar una los datos data 2d de forma parámetrica.

Presupone que la los datos contenidos en data2d son una imagen de la seccion central de algo con simetría cilíndrica, siendo el eje radial el eje x. Asumiendo esto, representa una superficie tridimensional mostrando en qué punto los datos llegan a sobrepasar cierto valor. De esta forma si la variable dependiente representara un potencial, el gráfico representaría una superficie equipotencial, de valor, el pasado como argumento: "threshold".
